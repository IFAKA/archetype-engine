# Generated Code

See [GENERATED_CODE.md](../../documentation/GENERATED_CODE.md) for the full documentation on generated code, including tests, API docs, and seed data.

Quick overview of what gets generated:
- Database schemas (Drizzle ORM)
- Validation schemas (Zod)
- API routers (tRPC)
- React hooks
- **Tests** (Vitest) - Comprehensive test suites
- **API Docs** (OpenAPI + Swagger) - Interactive documentation
- **Seed Data** - Realistic sample data

Run `npx archetype generate` to create all of this from your entity definitions.
